## Security Enhancements

1. **Organize Destination Validation** (handlers_batch.go:582-589)
   - Added isDirAllowed validation before processing organize requests
   - Prevents unauthorized file organization operations
   - Generic error message prevents path leakage

2. **Preview Destination Validation** (handlers_batch.go:659-666)
   - Added isDirAllowed validation before generating previews
   - Defense-in-depth for preview operations
   - Consistent security posture across endpoints

3. **Symlink Race Condition Fix** (handlers_batch.go:217-236)
   - Removed redundant per-file directory validation
   - Prevents TOCTOU race condition between EvalSymlinks calls
   - Added sanity check to ensure files are in scanned directory

## Reliability Improvements

4. **Scan Timeout Default** (handlers_batch.go:21-25, 189-192)
   - Added minScanTimeout constant (5 seconds)
   - Prevents immediate context cancellation when timeout is 0 or negative
   - Maintains backward compatibility for positive timeout values

5. **Effective Timeout Logging** (handlers_batch.go:203-206)
   - Logs actual timeout used instead of configured value
   - Improves debugging clarity when default timeout is applied
